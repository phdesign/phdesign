Win XP copy path to clipboard

posted on 23 January 2012 in uncategorized with 0 Comments

If you’re still stuck on Windows XP, you might find this script helpful. Save a shortcut to it in your ‘SendTo’ folder and simply right click on any file, select Send to > Copy Path and the full file path will be copied to your clipboard.

Requires clip.exe.
